How to add web maps to your portfolio website?


    Step 1. Define height and margins

    Our portfolio webpage has a title bar, which may hide the top portion of the maps and cause inproper display. So let's fix that by adding margins first. In addition, we will also add a fixed height for the map containers.

  • Find your portfolio folder (e.g., ges387). Edit the index.html file, copy the following lines and paste them inside the head section: 
    <style>
    body {
        height: 700px;
    }

    .item{
      margin-left:-10px;
      margin-right:-10px;
      height:700px;
    }

    @media (min-width: 1200px) {
      .item{
        margin-top: 40px;
      }
      .carousel-caption{
        bottom: 70px;
      }
    }

    @media (min-width: 992px) and (max-width: 1199px) {
      .item{
        margin-top: 90px;
      }
      .carousel-caption{
        bottom: 120px;
      }
    }

    @media (min-width: 768px) and (max-width: 991px){
      .item{
        margin-top: 90px;
      }
      .carousel-caption{
        bottom: 120px;
      }
    }

    @media (min-width: 488px) and (max-width: 767px) {
      .item{
        margin-top: 40px;
      }
      .carousel-caption{
        bottom: 70px;
      }
    }

    @media (max-width: 487px){
      .item{
        margin-top: 90px;
      }
      .carousel-caption{
        bottom: 120px;
      }
    }
</style>
    Specifically, we have added fixed height for the map containers to be 700px. We have also adjusted the top margin of the web maps to be added and the bottom margin of the captions.

    The lengthy code defines margins that will also automatically adjust based on browser window size so that your website can be displayed properly when viewed from tablet and/or smart phones (mobile-friendly).
